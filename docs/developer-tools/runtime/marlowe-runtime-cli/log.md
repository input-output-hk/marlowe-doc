title: Output the history of a contract
sidebar_position: 10

```console
Usage: marlowe-runtime-cli log [-t|--tail] [-c|--show-contract] CONTRACT_ID

  Display the history of a contract

Available options:
  -t,--tail                Await new events from the contract and print them as
                           they occur
  -c,--show-contract       Show the contract at each step
  CONTRACT_ID              The ID of the contract to log
  -h,--help                Show this help text
```
